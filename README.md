# CPlusPlus
主要面向程序员提升自己的C++写代码的水平。

使用CMake当做源码的管理编译系统，每个章节都有对应的序号，生成的可执行文件在build文件夹中。

第一章
快速地复习C++的基础知识，学习STL库使用。
智能指针的基础知识。
主要知识点：
01. helloworld程序
02. Namespaces 命名空间
03. Variables 变量
04. Literals 数字
05. Operators 操作符
06. Types 类型
07. Conditions 条件
08. Arrays 数组
09. Loops 循环
10. Functions 函数
11. Type Inference Part 1 类型推断
